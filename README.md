# DIO_Mapa_de_Habilidades
Mapa de habilidades Profissional com forte domínio em mercado bancário e financeiro, economia, crédito, contabilidade e sustentabilidade. Destaca-se pela capacidade analítica, interpretação de dados financeiros, visão estratégica de negócios, comunicação clara e atuação consultiva voltada a empresas, cooperativas e produtores.
